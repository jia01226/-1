# 明天回家 · 部署清单（照抄就行）

> 在服务器上执行（SSH 进去，进到 platform 目录）。

## ① 拉新代码 + 清老数据 + 重启
```bash
git pull
./venv/bin/python reset_app.py     # 清掉顾得时期的所有个人数据（推送订阅保留）
sudo systemctl restart gude
```

## ② 柯入驻
```bash
# 1. 把柯的人设放进来（从 kongkong 仓库拿，或先手写一份）
#    文件路径：platform/personas/柯.md
# 2. .env 里加一行：
CHARACTER=柯
# 3. 想改推送署名也顺手：
APP_NAME=柯
# 4. 再重启
sudo systemctl restart gude
```

## ③ 枕边日记 · 挂上夜班 cron
```bash
crontab -e
# 加一行（路径按你服务器实际的改）：
30 23 * * * cd /root/-1/platform && ./venv/bin/python diary_writer.py >> diary.log 2>&1
```

## ④ 手机上（不用电脑）
- 照 `快捷指令教程.md` 挂行踪上报（5 分钟）
- 打开 app → 日常页 → 看到「🌘 枕边日记」入口 = 新版部署成功
