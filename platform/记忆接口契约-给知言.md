## 鉴权(全站一致)

所有 `/api/*` 接口走同一个登录态(主页口令登录后的 session)。未登录返回 `401 {"error":"need_passcode"}` → UI 弹"回主页登录"门(inbox.html 里有现成写法)。

## 一、待确认收件箱(UI 顺序第 1 位)

### GET `/api/memory/pending`
待佳佳点头的卡(普通库+私密库合在一起)。返回数组:

```json
[{
  "id": 12,
  "type": "MEMORY",
  "content": "她最近在追一部老剧",
  "topic": "daily",
  "source": "ke_inferred",
  "importance": 3,
  "review_state": "pending_ke",
  "created_at": "2026-07-17 16:00:00",
  "store": "l2"
}]
```

- `store`:`"l2"`=普通记忆库 / `"private"`=私密库(UI 上标 🔒"私密·只单聊")。
- `source` 人话:`user_explicit`=你亲口说的 / `ke_inferred`=他记的 / `system_summary`=系统总结 / `legacy`=老记忆。

### POST `/api/memory/pending/review`
三键都走这一个接口:

```json
{"id": 12, "store": "l2", "approve": true}            // 确认保存
{"id": 12, "store": "l2", "approve": true, "content": "改好的内容"}  // 改一下再存
{"id": 12, "store": "l2", "approve": false}           // 不保存(直接丢弃,连向量一起清)
```

返回 `{"ok": true}`。**`store` 必须原样带回**(从 pending 列表里拿),不然会操作错表。

## 二、记忆卡片库(UI 顺序第 2 位)

### GET `/api/memory/cards?store=l2&status=active&q=樱粉`
参数全可选:
- `store`:`l2`(默认)/`private`
- `status`:`active`(默认,生效中)/`archived`(已归档)/`forgotten_buffer`(冷静期里)/`superseded`(被新版本替代)/`pending`(待确认)/`all`(全部)
- `q`:搜索词(搜内容和主题)

返回卡片数组(字段=表全列+`store`)。注意:**私密库结果里含 `scope="no_model"` 的卡**——那是"任何场景都不给模型看,仅后台可见"的档位,UI 上要标出来(如 🚫"只留档,他看不见"),别当普通卡展示。

### GET `/api/memory/card?id=12&store=l2`
卡片详情("为什么记住/何时被用过")。在卡片全字段之外多两个:

```json
{
  "id": 12, "content": "...", "source": "ke_inferred", "...": "...",
  "used_count": 5,
  "used": [{"at": "2026-07-17 16:23:58", "scope": "single", "query": "当时聊到的话"}]
}
```

`used`=最近被想起的记录(最多 20 条)。UI 呈现建议:"最近 5 次被想起 · 最近一次是 7/17 聊到「…」的时候"。

## 三、卡片纠错四件套(详情页里的操作)

| 操作 | 接口 | 请求体 | 人话文案建议 |
|---|---|---|---|
| 记错了,改内容 | POST `/api/memory/card/edit` | `{"id":12,"store":"l2","content":"改好的"}` | "改一下" |
| 归档 | POST `/api/memory/card/archive` | `{"id":12,"store":"l2"}` | "收进箱底"(留着,但他默认不再想起) |
| 忘掉 | POST `/api/memory/card/forget` | `{"id":12,"store":"l2"}` | "让他忘了这个"(**进七天冷静期,随时能反悔**) |
| 找回 | POST `/api/memory/card/recover` | `{"id":12,"store":"l2"}` | "捡回来" |
| 改可见范围 | POST `/api/memory/card/scope` | `{"id":12,"store":"l2","scope":"private"}` | 见对照表 |

- **edit 不是原地改**:后端会新开一张卡、旧卡转 `superseded` 留档(柯的硬规矩:重要记忆禁整条覆盖)。返回 `{"ok":true,"new_id":98}`,改完刷新列表即可。
- forget 的返回带一句 `note`:"已进七天冷静期,随时可找回"——可以直接当 toast 文案。
- scope 可选值:`l2` 库=`private/shared/group-safe/no_model`;`private` 库只有 `private/no_model`(传别的会被拒,返回 `{"ok":false}`)。
