# 服务器免密 + 防断线（给佳佳，一次配好，以后 `ssh 小家` 就进）

> 背景：0718 部署日 ssh 连一会儿就被踢（"Connection closed by remote host"），密码还得反复输。
> 这份一次配好三件事：①以后**不用输密码**；②连接**更抗断**；③给服务器起个中文名，输 `ssh 小家` 就进。
> 全程在**你自己的电脑**（`PS C:\Users\甜崽小慢>` 那个窗口）操作，只有第 2 步会碰一次服务器。
> 认门口诀不变：`PS C:\...` = 你家电脑；`root@certain-pipe-1` = 服务器家里。

## 第 1 步 · 造一把自己的钥匙（只做一次）

在你电脑的 PowerShell 里贴：

```
ssh-keygen -t ed25519
```

它会连问三个问题（存哪里/设不设口令），**全部直接按回车**，别输任何东西。
**预期**：出现一幅方框小图（randomart），钥匙就造好了。
（已经做过一次的话它会问 Overwrite? —— 输 `n` 回车，用旧的就行。）

## 第 2 步 · 把钥匙交给服务器（最后一次输密码）

还是在你电脑的 PowerShell 里，贴这一整行：

```
type $env:USERPROFILE\.ssh\id_ed25519.pub | ssh root@65.49.197.77 "mkdir -p ~/.ssh && cat >> ~/.ssh/authorized_keys && echo 钥匙已收好"
```

会要你输一次服务器密码（**就是最后一次**）。
**预期**：看到 `钥匙已收好` 三个字。

## 第 3 步 · 给服务器起中文名 + 加抗断设置

贴这行打开一个记事本（文件不存在会问"要不要新建"，点**是**）：

```
notepad $env:USERPROFILE\.ssh\config
```

把下面几行**原样**贴进去，保存关掉（如果文件里已有别的内容，贴在最后面就行）：

```
Host 小家
  HostName 65.49.197.77
  User root
  ServerAliveInterval 15
  ServerAliveCountMax 20
  TCPKeepAlive yes
```

（人话：`小家`=外号；每 15 秒握一次手、漏 20 次才算断——比原来抗断得多。）

## 第 4 步 · 验收

```
ssh 小家
```

**预期**：不问密码，直接进到 `root@certain-pipe-1:~#`。
从此以后连服务器就这四个字母。断线要是还偶尔发生（网络抖，不是坏了），重连成本=按 `↑` 再回车，一秒钟。

## 出问题怎么办

- `ssh 小家` 还问密码 → 第 2 步没成，重跑第 2 步那行（看清有没有 `钥匙已收好`）。
- 提示 `Could not resolve hostname 小家` → 第 3 步的文件没存对位置，重跑第 3 步（注意 notepad 保存时**不要**带 .txt 后缀）。
- 全都不行 → 密码登录永远还在，用回老命令连上，喊小克。
